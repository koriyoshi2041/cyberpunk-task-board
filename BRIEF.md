# 任务管理看板 - 浅色赛博朋克风格

## 项目概述
一个带有趣味微交互的任务管理看板，采用浅色系赛博朋克视觉风格。

## 技术栈（必须使用）
- **框架**: Vite + React + TypeScript
- **动画**: GSAP + ScrollTrigger
- **平滑滚动**: Lenis (@studio-freight/lenis)
- **样式**: Tailwind CSS
- **状态**: Zustand

## 视觉风格：浅色赛博朋克

### 配色方案
```css
:root {
  /* 主背景 - 极浅的科技灰/米白 */
  --bg-primary: #f8f9fa;
  --bg-secondary: #eef1f5;
  
  /* 赛博朋克强调色 */
  --cyber-pink: #ff2d95;
  --cyber-cyan: #00fff0;
  --cyber-purple: #9d4edd;
  --cyber-yellow: #ffe66d;
  
  /* 霓虹发光效果 */
  --glow-pink: 0 0 20px rgba(255, 45, 149, 0.4);
  --glow-cyan: 0 0 20px rgba(0, 255, 240, 0.4);
  
  /* 文字 */
  --text-primary: #1a1a2e;
  --text-secondary: #4a4a6a;
}
```

### 视觉特征
1. **霓虹边框/发光** - 卡片边缘有微妙的霓虹发光效果
2. **网格线背景** - 浅色网格线装饰（赛博朋克经典元素）
3. **故障效果(Glitch)** - 文字偶尔有轻微故障抖动
4. **扫描线** - 微妙的 CRT 扫描线覆盖层
5. **像素化装饰** - 角落的像素化装饰元素

## 微交互设计（重点！）

### 1. 自定义光标
- 默认：小圆点 + 外圈跟随
- 悬停任务卡：放大 + 显示 "DRAG"
- 悬停按钮：磁性吸附效果

### 2. 任务卡片
- **拖拽时**：卡片轻微旋转 + 发光增强 + 投影加深
- **悬停时**：霓虹边框亮起 + 内容微微上浮
- **完成时**：故障闪烁 + 粒子爆炸效果 → 卡片消失

### 3. 列标题
- 悬停时文字有轻微 glitch 抖动
- 任务数量变化时数字有翻转动画

### 4. 添加任务按钮
- 点击时：按钮展开为输入框
- 脉冲呼吸效果（idle 状态）

### 5. 全局效果
- 页面加载：扫描线从上到下 + 元素依次 reveal
- 背景：缓慢移动的网格线
- 角落：闪烁的数据装饰（fake data readouts）

### 6. 声音反馈（可选）
- 卡片拖拽：轻微电子音
- 任务完成：成功音效
